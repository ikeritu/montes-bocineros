# INFORME V4.11A — Biblioteca documental unificada

## Decisión

Se aplica la opción limpia: `biblioteca.html` pasa a ser una página única que absorbe las antiguas páginas documentales auxiliares:

- `cadena-trueba.html`
- `barrio-banales.html`
- `citas.html`
- `archivo-tecnico.html`
- `informes.html`
- `pendientes-documentales.html`

## Criterio de fusión

La nueva página no intenta conservar todo el detalle narrativo de las páginas antiguas. Filtra duplicados, reduce paja y conserva los datos de control:

- 1342, 1394 y 1452 como capa de vozinas, Junta, merindades y oficios.
- Lope, Goicolea e Ibargüen-Cachopín como capa legendaria/cultura sonora.
- Madoz 1847 como fuente puente de heraldos en alturas.
- Trueba 1858 como lectura montañera flexible de siete/siete.
- Trueba 1862 como eslabón intermedio: cinco vocinas/cinco montes sin nombres.
- Trueba 1872 como primer punto firme localizado para la lista nominal completa.
- 1873/1880 como recepción posterior y variantes ampliadas.
- Barrio/Bañales como estudio moderno central para separar rito, leyenda, recepción y fijación decimonónica.

## Componentes UX integrados

- Buscador por texto.
- Filtros por capa documental.
- Línea de tiempo interactiva.
- Acordeón “Mito vs. realidad”.
- Tarjetas documentales.
- Tabla de citas verificadas.
- Quiz final de 3 preguntas.

## No incluido en esta fase

No se fusionan todavía:

- `metodologia.html`
- `citar.html`
- `afirmaciones.html`
- `autor.html`
- `personajes.html`

Eso queda para V4.11B y V4.11C.
