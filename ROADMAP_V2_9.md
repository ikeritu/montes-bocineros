# ROADMAP V2.9 — Cronología interactiva

## Objetivo
Convertir la cronología documental en una herramienta más interactiva y didáctica sin perder rigor ni citabilidad.

## Aplicado
- [x] Sustituir la timeline vertical como elemento principal por un eje horizontal clicable.
- [x] Añadir ficha dinámica del hito seleccionado.
- [x] Mantener filtros por tipo de evidencia.
- [x] Añadir navegación anterior/siguiente para móvil.
- [x] Conservar lista completa como fallback accesible y rastreable.
- [x] Evitar librerías externas y mantener GitHub Pages ligero.

## No aplicado
- [ ] Animaciones complejas tipo app.
- [ ] Audio o recreaciones sonoras.
- [ ] Nuevos hitos no verificados.
- [ ] Llorente como hecho cerrado.

## QA recomendado
Revisar `cronologia.html` en escritorio y móvil:
- filtros;
- clic en puntos;
- anterior/siguiente;
- lista fallback;
- enlaces a fuentes;
- comportamiento sin errores JS.
