# ROADMAP V3.4 — Rediseño visual integral aplicado

## Objetivo
Aplicar de una vez los cambios visuales propuestos sin destruir páginas documentales citables.

## Cerrado en esta versión
- Portada rediseñada con lenguaje visual del prototipo: hero oscuro, ecos, contadores, filtros de certeza y ruta de lectura.
- Bloque de capas de certeza en portada para explicar documentado / plausible / recepción / no afirmable.
- Visual acústica hipotética integrada con cautela metodológica explícita.
- `personajes.html` convertido en un “Quién es quién” interactivo: filtros por rol, tarjetas compactas y panel de ficha seleccionada.
- Mejorado el cierre “Tres caminos para leer y uno para comprobar”, con acceso directo al Archivo documental.

## Reglas mantenidas
- No se altera la tesis histórica.
- No se presenta Amboto como quinto monte tradicional.
- La visual acústica no se presenta como prueba histórica ni simulación científica.
- Las páginas profundas siguen vivas y citables.
