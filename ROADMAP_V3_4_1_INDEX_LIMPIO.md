# V3.4.1 — Limpieza visual de portada

## Objetivo

Limpiar `index.html` para que funcione como portada clara y no como página larga de demostración interactiva.

## Cambios

- Mantener hero oscuro con mensaje central.
- Mantener bloque “Tres caminos para leer y uno para comprobar”.
- Mantener footer oscuro con archivo y apoyo.
- Retirar de la home las secciones largas de capas de certeza, visual acústica, cita y apoyo independiente.
- Mantener esas piezas en páginas internas (`guia-lector.html`, `montes.html`, `personajes.html`) para no perder valor didáctico.

## Criterio

La portada debe cargar rápido, explicar la tesis en segundos y llevar al lector a cuatro destinos claros: Historia, Montes y mapa, Síntesis crítica y Archivo documental.
